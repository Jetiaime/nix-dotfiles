# PROMPTING-EXAMPLES.md - Example Prompts

Common prompts to get the most out of TeAmo.

## Quick Actions

### File Operations

```
"读一下 ~/Documents/notes.txt 的内容"
"找一下 src 目录下所有的 .ts 文件"
"创建一个 todo.md 文件，内容是..."
```

### System Commands

```
"看一下 CPU 使用率"
"有哪些进程在运行"
"查看最近的系统日志"
"磁盘空间还剩多少"
```

### Web & Research

```
"总结一下这篇文章: https://example.com/article"
"搜索一下最近的 Nix 更新"
"帮我查一下这个库的文档"
```

### Screenshots (peekaboo)

```
"截个屏看看现在屏幕上是什么"
"帮我截图并描述一下当前界面"
```

## Development Tasks

### Code Help

```
"帮我看看这个函数有什么问题"
"解释一下这段代码在做什么"
"给这个模块写测试"
"重构一下这个函数，让它更清晰"
```

### Nix / nix-darwin

```
"帮我添加一个新的 homebrew cask"
"看一下当前的 nix-darwin 配置"
"帮我更新 flake inputs"
"添加一个新的 OpenClaw 插件"
```

### Git

```
"看一下 git status"
"帮我写个 commit message"
"最近的提交记录是什么"
```

## Safety Patterns

```
"先 dry-run 看看效果"
"别删除任何东西，只列出来"
"执行 sudo 之前先确认"
"先告诉我你打算怎么做，再动手"
```

## System Management

```
"检查一下 OpenClaw gateway 的状态"
"重启一下 gateway 服务"
"看看有没有什么错误日志"
"帮我 rebuild 一下 nix-darwin"
```
